List 和 数组之间的转换

一、数组转为 List

1. 方法一

1
1
List<Element> arrayList = new ArrayList<Element>(Arrays.asList(array));

2. 方法二

1
1
List<Element> list = Arrays.asList(array);

这不是最好的，因为asList()返回的列表的大小是固定的。事实上，返回的列表不是java.util.ArrayList，而是定义在java.util.Arrays中一个私有静态类，是由原始数组支持的固定大小的列表。如果添加或删除列表中的元素，程序会抛出异常UnsupportedOperationException。

3. 方法三

3
1
Element[] array = {new Element(1), new Element(2)};
2
List<element> list = new ArrayList<element>(array.length);
3
Collections.addAll(list, array);

2
1
List<String> list = new ArrayList<>();
2
String[] array = list.toArray(new String[0]);
